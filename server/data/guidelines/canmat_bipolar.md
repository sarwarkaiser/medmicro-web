# Management of Acute Mania

## First-line Agents
- Lithium
- Quetiapine
- Divalproex
- Asenapine
- Aripiprazole
- Paliperidone
- Risperidone
- Cariprazine

## Second-line Agents
- Olanzapine
- Carbamazepine
- Ziprasidone
- Haloperidol

# Management of Bipolar Depression

## First-line Agents
- Quetiapine
- Lurasidone + Lithium/Divalproex
- Lithium (monotherapy)
- Lamotrigine (maintenance/adjunct)

## Second-line Agents
- Divalproex
- SSRIs (adjunct ONLY)
